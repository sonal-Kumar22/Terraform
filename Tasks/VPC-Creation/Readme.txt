Generate key-pair value using command :  ssh-keygen 

Commnad to run in the location/direcotry where these terraform files are stored - run the command :ssh-keygen -f ec2-key-pair
this command will create public and private keys with ec2-key-pair name in the directory.
Ex: ec2-key-pair.pub --> public key
    ec2-key-pair --> private key
